Historical experiment and one-off operator scripts live here.

Why this directory exists:
- These scripts capture past experiment workflows that are no longer supported
  as active `scripts/` entrypoints.
- They remain useful as historical reference when we need to reconstruct how a
  benchmark or manual review process used to work.

How to use it:
- Treat these files as historical context, not as current supported tooling.
- If we revive one of these workflows, move it back into `scripts/` with fresh
  docs and tests instead of calling the archived path a supported entrypoint.
